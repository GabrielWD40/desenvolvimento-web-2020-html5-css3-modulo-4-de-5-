Anotações sobre a auala:

Nessa aula, nós vimos algumas melhorias muito legais sobre os formulários em HTML. Vimos

  • Como deixar um campo com preenchimento obrigatório: REQUIRED
  • Como impor um tamanho mínimo/máximo de caracteres: MIN-LENGTH e MAX-LENGTH
  • Como colocar um input tipo PASSAWORD
  • Como colocar um botão que limpa todos os campos do formulário: INPUT RESET
  • Atributo SIZE:Mostra quantos caracteres o input vai mostrar por vez
  • Tag placeholder: Vai dentro das inputs e fornece uma "dica" do que é para ser preenchido naquele lugar.
  • autocomplete dentro de inputs: Aqui você pode dizer que tipo de input é esse, se é de senha, nome de usuário... Isso ajuda o autocomplete do navegador e facilita pro usuário.

Nós colocamos esses atributos dentro dos inputs, ok? SEMPRE DENTRO DOS INPUTS.
Repare que se você tentar enviar um campo que esteja com REQUIRED, ele não vai deixar e pedirá para preenche-lo.
Além disso, ele vai dar um aviso quando o usuario digitar nomes acima ou abaixo do limite de caracteres preestabelecidos.
Repare que quando utilizmos o atributo size dentro das inputs, o tamanho visual delas é diminuido e, a quantidade de caracteres que ela mostra também é menor. Ele cria uma rolagem lateral para quando o dado for maior...
Uma coisa bem legal que vimos é sobre o autocomplete dentro das inputs. Você precisa estar com o autocomplete habilitado dentro da tag form para isso. Depois, você vai dentro duma input e colocar o autocomplete e o tipo dela, o navegador vai recomendar com base no histórico do usuário.

Nessa aula vimos que a tag input tem muitos controles, mas no código, tem vários atribuos que nospermitem controlar melhor a entrada de dados.
