ANOTAÇÕES DAS AULAS:

Nessa aula, vamos falar sobre os atributos input tipo checkbox e input tipo radio. Elas são bem úteis e amplamente usadas em formulários.
O tipo radio só permite escolhermos uma opção. Já o modo chheckbox permite uma seleção multiplas de opções.

Outra coisa que devemos lembrar é de sempre colocar os labels para conseguirmos identificar as opções ofertadas ao usuário.
Sempre que você for usar as inputs do tipo radio, todas as opções devem ter o mesmo nome. Isso serve para ele reconhecer que todas elas pertecem ao mesmo grupo e funcionarem corretamente.
Além disso, ainda é possível nós conseguirmos já deixar opções pré-selecionadas com o atributo checked
