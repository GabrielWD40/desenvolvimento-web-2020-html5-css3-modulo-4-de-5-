Anotações da aula:

Na aula passada, nós vimos um pouco sobre as mídias types, que são estilos para telas, impressões e o tipo all, por exemplo. Não existe mídia type para os tamanhos de tela, e sim mídia type.

mídia query é = média type(tipo de mídia) + media features(tamanho de mídia em diferentes telas)

Se uma tela é mais comprida do que alta, ou seja, a largura for maior que a a altura, ela está em modo paisagem.
Não obstante, se uma tela é mais alta do que larga, ela está em modo retrato. 

    Uma coisa muito bacana que nós apredemedos foi o seguinte: 
        
        Na chamada da média query lá no cabeçalho da página, é possível chamar mais de um estilo para cada media query. 
        Além disso, também é possível informar a orientação (posição relativa) da tela do seu visitante.

        Veja um exemplo de código em HTML:
        
        
        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta http-equiv="X-UA-Compatible" content="ie=edge">
        <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
        
        <link rel="stylesheet" href="./estilos/all.css" media="all">
            
O link de estilo acima irá servir de modelo para todos os tipos de dispositivos, suas regras serão globais. 

            Por isso, colocamos media="all";


        <link rel="stylesheet" href="estilos/retrato.css" media="screen and (orientation: portrait)">
            
O link de estilo acima irá servir apenas para telas com orientação em retrato. 

            Por isso, colocamos o media features entre parênteses. 

        
        <link rel="stylesheet" href="estilos/paisagem.css" media="screen and (orientation: landscape)">
        
        
        <title>Masdasdases</title>
        </head>
        
        
        Na hora de declarar uma media query é possível informar o media type e o média features.
        Só que tem um detalhe, as médias features precisam estar entre parênteses. Exemplo:

        <link rel="stylesheet" href="caminho-do-aquivo/nome-do-arquivo.css" media="screen and (orientation: landscape or portrait)">
        A sintax para isso é a seguinte: media="tipo de mídia and(orientation: orientação da tela do visitante.)"

        OBS: Para cada mídia feature você precisará colocar um parênteses.
