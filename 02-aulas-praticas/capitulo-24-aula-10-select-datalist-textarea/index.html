ANOTAÇÕES DA AULA:

Nessa aula nós falamos sobre uma coisa que eu queria aprender há muito tempo: os formulários com várias opções.
Vimos que para usar uma lista de opções pré-definidas pelo desenvolvedor, é necessário usar: 

        Você tem que usar a tag <SELECT> → <OPTION>, só assim vai funcionar. 
        Você pode usar um agrupador de opções, organizando melhor o nosso código. USE A TAG <OPTGROUP>
        Dentro da tag <optgroup> você pode colocar uma LABEL, ela será o nome daquele agrupamento.

        EXEMPLO DE LISTA PRÉESTABELECIDA:  
        <label for="Icomidas">
        <select name="comidas" id="Icomidas">
            <optgroup label="Nome do agrupamento"> → organiza as opções em grupos.
                <option value="alcatra"> Alcatra </option>
                <option value="bife"> Bifão  </option>
                <option value="costela"> Costela  </option>
                <option value="Frango"> Franguin  </option>
                <option value="linguiça"> Linguiça  </option>
            <optgroup>
            <optgroup label="frutas"> 
                <option value="Banana"> Banana </option>
                <option value="Maça"> Maça </option>
                <option value="Melancia"> Melancia </option>
                <option value="Ponkan"> Ponkan </option>
                <option value="Morango"> Morango </option>
            </optgroup>

        </select>


        EXEMPLO DE INPUT DE TEXTO COM OPÇÕES:
        <label for"ilista-de-profissao">Digite a sua profissão: </label>
        <input type="text" name="lista-de-profissao" id="ilista-de-profissao" placeholder="Ex: Frentista" list="nome de lista de sugestões">
        
        <datalist id="ID DO ATRIBUTO LIST DO INPUT DESEJADO">
            <option value="valor">nome da opção</option>
            <option value="desocupado">Estudante</option>
            <option value="vadio">Vagabundo</option>
            <option value="artigo 171">Do corre</option>
        </datalist>

            *Pra usar essa opção com sugestões de opções, você precisará usar o datalist e dentro dele, informar o ID do atributo list que está dentro da input que você deseja ver essas opções.
            *Dentro da tag <datalist>, use as tags  <option value="">  
    

        INPUT PARA DIGITAR TEXTO:
        <label for="imsg">Digite uma mensagem: </label>
        <textarea name="msg" id="imsg" cols="50" rows="10">   </textarea>
            *Nesse exemplo, o atributo itpa abrir um caixa de texto onde o usuário poderá digitar textos.
            *É parecido com aqueles campos de denúncias dos jogos, aonde você descreve um problema por meio de texto e envia um ticket ao suporte.

            *ROWS é o tamanho das linhas, mas elas são infinitas, ok?
            *COLS é o tamanho da largura da caixa de texto, quantas colunas ela vai ter. Quanto menor, menor o comprimento da caixa.
