Anotações sobre a aula

Vimos que os fourmulários possuem 2 métodos: GET e POST.
O método GET permite que os dados inseridos nas nas inputs sejam vistos na URL, Já o método POST oculta esses dados da URL.
ENTRETANTO: NENHUM DELES É SEGURO! Ambos usam protocolo HTPP.
Por padrão, os formulários vem com o método GET, mas é bem simples de mudar. Basta fazer: 


        < form action="" method="get" → mude para 'post'... autocomplete="off" > 
        
        </form>7

MAS, QUANDO USAR CADA TIPO??

Método GET → Use quando for coletar dados que não sejam sensíveis: Nome, peso, altura, cor preferida, nome do cachorro... BObeiras que não prejudiquem o usuário

Método POST → Use para dados sensíveis, como senhas, endereços, fotos, cartões de crédito e coisas que se vazarem, podem foder com a vida de alguém.

Curiosidade:

Se você enviar dados por um formulário usando o método GET, você só pode envia até 3mil bytes.É como se fosse três mil letras.
