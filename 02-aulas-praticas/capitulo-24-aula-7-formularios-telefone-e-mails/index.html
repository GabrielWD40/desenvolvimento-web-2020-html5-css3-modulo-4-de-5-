Anotações sobre a aula:

Nessa aula nós falamos sobre os campos de telefone e email dentro de um formulário. Para usar telefones, use o input tipo tel e com os e-mails, use o tipo e-mail.
Se você quiser usar uma input do tipo telefone, use o input tipo telefone e depois use o G

Vimos que no campo telefone, é possível usar uma propriedade chamada de REGEX para definirmos o estilo padrão de como os dados se comportarão.
Sabe quando queremos que o modelo fique (xx) x xxxx - xxxx? Então, o RegEx serve para isso. Dentro do input, devemos colocar a propriedade pattern=""

O RegEx funciona da seguinte maneira: 


        entre colchetes [ ] → Você coloca o intervalo inicial - intervalo final. Ex: [a - z], [0 - 9]... 
            Acima, quero dizer que o intervalo irá de 'a' até 'z' ou então, de '0' até '9'.
        
        Depois, entre chaves { } → Você coloca a quantidade de vezes que o campo terá. Ex: [0-9]{5}
            Acima, estou querendo dizer que o campo em questão terá cinco números de 0 à 9.


        No caso de uma input detelefone, eu faria o seguinte: 

        <label for="itelefone">Preencha o seu número: </label>
        <input type="tel" name="telefone" id="itelefone" required autocomplete="tel" pattern:^[0-9]{2} [0-9]{5}-[0-9]{4} $ placeholder="52 98765-4321" >

        Isso seria útil para orientar o usuário!
        Lembrete: use '^' para simbolizar o início da expressão e '$' para simbolizar o fim dela.

OUTRA COISA LEGAL: O agrupamento de recolhimento de informações com a tag FIELDSET
Olha essa borda que foi feita em volta do nosso formulário, ela é o FIELDSET. Ela agrupa elementos com a mesma finalidade, aquele escritinho ali é dado pela tag <legend> dentro do <fieldset>
